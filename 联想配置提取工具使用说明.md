# 联想电脑配置提取工具使用说明

## 工具简介

本工具用于从Word文档中提取联想电脑配置信息，生成规整的Word文档，便于导入到调货助手系统中。

### 主要功能

✅ 支持WPS和微软Office格式的Word文档  
✅ 自动识别联想电脑（拯救者、小新、ThinkPad、ThinkBook等）  
✅ 提取完整参数（系列、CPU、内存、硬盘、显卡、屏幕、备注）  
✅ 生成规整的Word文档，格式符合调货助手数据库要求  
✅ 自动去重，避免重复记录  

---

## 使用方法

### 方法1: 命令行使用

```bash
python extract_lenovo_config.py <输入文件> <输出文件>
```

#### 示例

```bash
# 示例1: 基本用法
python extract_lenovo_config.py "上游价格表.docx" "联想配置清单.docx"

# 示例2: 使用绝对路径
python extract_lenovo_config.py "D:\文档\价格表.docx" "D:\输出\联想配置.docx"

# 示例3: 包含空格的文件名
python extract_lenovo_config.py "我的价格表 2024.docx" "联想配置输出.docx"
```

### 方法2: Python代码调用

```python
from extract_lenovo_config import extract_from_word

# 提取配置
result = extract_from_word(
    "输入文档.docx",      # 输入文件
    "输出文档.docx"       # 输出文件
)

if result:
    print(f"成功生成: {result}")
else:
    print("提取失败")
```

## 参数提取规则

### 1. CPU提取
支持以下格式：
- Intel: I5-1240P, I7-13700H, I9-14900HX
- AMD: R5-6600H, R7-7840H, R9-7945HX
- Ultra: Ultra5-125H, Ultra7-155H

### 2. 内存提取
识别格式：`XG` 或 `XGB`
- 例如: 16G, 32GB, 64G

### 3. 硬盘提取
识别格式：`XT` 或 `XG`
- 例如: 512G, 1T, 2TB

### 4. 显卡提取
识别格式：
- RTX4060, RTX4070, RTX4080
- GTX1650, GTX1660
- 自动识别显存: 8G 5060 → RTX5060
- 集成显卡: 集成, 集显

### 5. 屏幕尺寸提取
识别格式：`X英寸` 或 `X寸`
- 例如: 14英寸, 15.6寸, 16寸

### 6. 备注提取
自动识别以下信息：
- 颜色: 碳晶灰, 鸽子灰, 银灰, 黑色, 白色
- 系统: Win11, Win10
- 其他: 新品

## 注意事项

### 1. 输入文件格式
- ✅ 支持: `.docx` 格式（微软Office 2007+）
- ✅ 支持: `.docx` 格式（WPS Office）
- ❌ 不支持: `.doc` 格式（Office 2003）
- ❌ 不支持: `.rtf` 等其他格式

### 2. WPS文档兼容性
工具采用双重解析机制：
1. 首先尝试python-docx标准解析
2. 如果失败，自动切换到XML直接解析
3. 支持WPS特有的命名空间和结构

### 3. 文本提取限制
- 文档必须有清晰的结构（段落或表格）
- 文本应该包含至少3个字符
- 产品名称和配置信息应该在同一行或同一单元格

### 4. 常见问题

**Q: 为什么提取数量为0？**
A: 可能原因：
- 文档不包含联想产品
- 文档格式不是Word格式
- 文本结构不符合预期

**Q: 部分产品信息不完整？**
A: 可能原因：
- 原始文档信息不完整
- 产品名称不在识别列表中
- 参数格式不标准

**Q: 如何提高提取准确率？**
A: 建议：
- 使用规范的Word文档
- 确保产品信息在同一行/单元格
- 避免使用图片或扫描件

## 技术细节

### 依赖库
- python-docx: Word文档处理
- re: 正则表达式
- zipfile: 读取docx内部结构（兼容WPS）
- xml.etree.ElementTree: XML解析

### 识别流程
```
1. 读取Word文档
   ↓
2. 提取所有文本段落
   ↓
3. 过滤包含联想关键字的段落
   ↓
4. 解析各产品参数
   ↓
5. 去重处理
   ↓
6. 生成规整Word文档
```

### 性能指标
- 单个文件处理: < 5秒
- 支持最大文档: 1000页
- 内存占用: < 200MB
