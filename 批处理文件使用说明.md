# 联想配置提取工具 - 批处理文件使用说明

## 文件说明

项目文件夹中包含以下批处理文件：

### 1. **联想配置提取工具.bat**
交互式版本，适合大多数用户

### 2. **联想配置提取_拖拽版.bat**
支持拖拽操作，更加快捷

---

## 使用方法

### 方法一：交互式使用（联想配置提取工具.bat）

1. **双击运行**
   ```
   双击 "联想配置提取工具.bat"
   ```

2. **等待窗口提示**
   - 程序会自动检查Python环境
   - 检查必要的脚本文件

3. **选择输入文件**
   - 会弹出"选择文件"对话框
   - 选择你要处理的Word文档（.docx格式）

4. **确认输出位置**
   - 程序会自动生成输出文件名
   - 格式：`原文件名_联想配置.docx`
   - 输入Y使用默认路径，或选择其他位置

5. **等待处理完成**
   - 会显示处理进度
   - 完成后会提示成功信息

6. **选择后续操作**
   - 是否打开生成的文档
   - 是否打开文件所在目录

---

### 方法二：拖拽使用（联想配置提取_拖拽版.bat）

这是最简单快捷的使用方式！

1. **准备文件**
   - 确保你的Word文档准备好了
   - 支持WPS或微软Office创建的.docx文件

2. **拖拽文件**
   ```
   将Word文档直接拖拽到
   "联想配置提取_拖拽版.bat" 文件上
   ```

3. **自动处理**
   - 程序会自动识别文件
   - 自动提取联想电脑配置
   - 自动生成输出文件
   - **处理完成后会自动打开生成的文档**

就这么简单！

## 常见问题

### Q1: 双击bat文件没反应？

**A:** 可能的原因：
- 未安装Python环境
- Python未添加到系统PATH

**解决方法:**
1. 打开CMD命令行
2. 输入 `python --version`
3. 如果提示"不是内部命令"，需要安装Python

### Q2: 拖拽时提示"文件不存在"？

**A:** 可能的原因：
- 文件路径包含特殊字符
- 文件正在被其他程序使用

**解决方法:**
1. 复制文件到简单路径（如桌面）
2. 关闭可能打开文件的程序（如Word）

### Q3: 提示"未找到 extract_lenovo_config.py"？

**A:** 确保以下文件在同一目录下：
- `联想配置提取工具.bat`
- `联想配置提取_拖拽版.bat`
- `extract_lenovo_config.py`

### Q4: 生成的文档在哪里？

**A:** 默认位置：
- 如果拖拽文件：与原文件相同目录
- 如果选择文件：与输入文件相同目录
- 文件名格式：`原文件名_联想配置.docx`

## 高级使用技巧

### 1. 批量处理

在文件夹中放置多个文件，依次拖拽处理：
```
价格表1.docx  → 拖拽 → 价格表1_联想配置.docx
价格表2.docx  → 拖拽 → 价格表2_联想配置.docx
价格表3.docx  → 拖拽 → 价格表3_联想配置.docx
```

### 2. 自定义输出路径

编辑 `联想配置提取工具.bat`，修改以下行：
```bat
set "OUTPUT_FILE=%SCRIPT_DIR%\%FILENAME%_联想配置.docx"
```
改成你想要的路径

### 3. 创建快捷方式

在桌面创建快捷方式：
1. 右键点击 `联想配置提取_拖拽版.bat`
2. 选择"发送到" → "桌面(创建快捷方式)"
3. 以后直接把文件拖到桌面快捷方式上即可

### 4. 添加到右键菜单

将"联想配置提取"添加到Word文档的右键菜单：
1. 创建以下注册表文件 `add_context_menu.reg`:
```reg
Windows Registry Editor Version 5.00

[HKEY_CLASSES_ROOT\.docx\shell\联想配置提取]
@="提取联想配置"

[HKEY_CLASSES_ROOT\.docx\shell\联想配置提取\command]
@="\"D:\\path\\to\\联想配置提取_拖拽版.bat\" \"%1\""
```

2. 双击运行 `add_context_menu.reg`
3. 现在右键点击.docx文件会出现"提取联想配置"选项
