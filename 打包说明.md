
# 调货助手 - 打包成 EXE 说明

## 方案一：升级 PyInstaller 后打包（推荐）

以管理员身份运行 PowerShell 或 CMD，执行以下命令：

```bash
pip install --upgrade pyinstaller pyinstaller-hooks-contrib
```

然后在项目目录下运行：

```bash
pyinstaller --onefile --windowed --name "调货助手" ^
  --hidden-import PyQt6 ^
  --hidden-import PyQt6.QtCore ^
  --hidden-import PyQt6.QtGui ^
  --hidden-import PyQt6.QtWidgets ^
  --hidden-import openpyxl ^
  --hidden-import docx ^
  --hidden-import PIL ^
  run.py
```

打包完成后，EXE 文件会在 `dist/` 文件夹中。

---

## 方案二：使用 Nuitka 打包（性能更好）

Nuitka 可以把 Python 编译成真正的机器码，体积小，运行快。

安装 Nuitka：
```bash
pip install nuitka zstandard ordered-set
```

打包：
```bash
python -m nuitka --standalone --onefile --windows-disable-console --plugin-enable=pyqt6 --output-filename="调货助手.exe" run.py
```

---

## 方案三：便携版（最简单，不用打包）

创建一个批处理文件 `启动调货助手.bat`，内容如下：

```batch
@echo off
chcp 65001 >nul
echo 正在启动调货助手...
python run.py
pause
```

这样用户只需要安装一次 Python，然后双击这个 bat 文件就可以了。

## 打包后的使用说明

打包好的 EXE 是单文件版，直接双击运行就可以了。

第一次运行时会在同级目录下创建 `data/` 文件夹存放数据库。
