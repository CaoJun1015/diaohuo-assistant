## 🔧 常见问题及解决

### 问题1：CMD一闪而过

**原因**：执行出错但窗口自动关闭

**解决方法**：
1. 使用"调试版"查看具体错误
2. 运行"诊断Python环境.bat"
3. 根据提示解决

---

### 问题2：Python未安装

**表现**：
```
'python' 不是内部或外部命令
```

**解决方法**：
1. 下载Python：https://www.python.org/downloads/
2. 安装时**务必勾选** "Add Python to PATH"
3. 完成后重新运行

---

### 问题3：python-docx未安装

**表现**：
```
No module named 'docx'
```

**解决方法**：
运行以下命令：
```cmd
pip install python-docx
```

或在"调试版"中会自动提示安装

---

### 问题4：找不到文件

**表现**：
```
extract_lenovo_config.py 不存在
```

**解决方法**：
确保以下文件在同一目录：
- 联想配置提取工具_调试版.bat
- extract_lenovo_config.py

## 💡 高级技巧

### 1. 创建桌面快捷方式

```
1. 右键"联想配置提取_拖拽版.bat"
2. 选择"发送到" → "桌面(创建快捷方式)"
3. 以后直接把文档拖到桌面快捷方式即可
```

### 2. 添加到右键菜单

创建文件 `add_menu.reg`：
```reg
Windows Registry Editor Version 5.00

[HKEY_CLASSES_ROOT\.docx\shell\联想提取]
@="提取联想配置"

[HKEY_CLASSES_ROOT\.docx\shell\联想提取\command]
@="\"D:\\path\\to\\联想配置提取_拖拽版.bat\" \"%1\""
```

双击运行即可添加右键菜单

### 3. 批量处理

创建一个批处理文件 `批量处理.bat`：
```bat
@echo off
cd /d "%~dp0"

for %%f in (*.docx) do (
    python extract_lenovo_config.py "%%f" "%%~nf_联想配置.docx"
)

pause
```
